# Friburgo Geek 2026

Site oficial do **Friburgo Geek 2026**, o maior evento geek de Nova Friburgo, produzido pela Alucard Animes Produções.

- Site: https://www.friburgogeek.com.br
- Produtora: https://www.alucardanimes.com.br

## Stack

- Vite
- TypeScript
- React
- Tailwind CSS
- shadcn-ui

## Desenvolvimento local

Requisitos: Node.js e npm.

```sh
# 1. Clonar o repositório
git clone <YOUR_GIT_URL>

# 2. Entrar na pasta do projeto
cd <YOUR_PROJECT_NAME>

# 3. Instalar dependências
npm install

# 4. Rodar em modo de desenvolvimento
npm run dev
```
